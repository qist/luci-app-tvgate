<%:TVGate%>

<%:Overview%>

<%:TVGate is a high-performance local network resource forwarding and proxy tool.%>

  • <%:Forward internal network resources (such as http, https, rtsp, rtp) to the external network through HTTP%>
  • <%:Support multicast, unicast audio and video streams converted to HTTP accessible links%>
  • <%:Built-in Web management interface for visual configuration%>
  • <%:Support Docker deployment, systemd service management and OpenWrt system integration%>

<%:How to use%>

  1. <%:Configure the download proxy (e.g., https://hk.gh-proxy.com/) to speed up downloads%>
  2. <%:Click "Download/Update Binary" to download the TVGate executable%>
  3. <%:Enable the service and configure the listening ports%>
  4. <%:Access the web interface at http://[router-ip]:[config-port] to manage settings%>
  5. <%:Access IPTV streams at http://[router-ip]:[streaming-port]/channel-url%>
